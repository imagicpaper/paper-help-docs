目前文本编辑器，支持段落设置（章节标题、无序列表）、字体设置（加粗、斜体、下划线、删除线、颜色）。

## 段落设置

支持 MarkDown 标记语言风格，使用“#”+“空格”设置段落标题，使用“-”+“空格”添加无序列表。
